pytket-quantinuum
==================================

.. image:: CQCLogo.png
   :width: 120px
   :align: right

``pytket-qujax`` is an extension to ``pytket`` that allows ``pytket`` circuits to
be handed over to qujax.

``pytket-qujax`` is available for Python 3.8, 3.9 and 3.10, on Linux, MacOS
and Windows. To install, run:

::

    pip install pytket-qujax

.. note::   Using the ``QuantinuumBackend`` requires a `Quantinuum <https://www.quantinuum.com/>`_ account.
            The user will be prompted for their login credentials when making API calls such as calling the ``process_circuits`` method or querying ``backend_info``.
