pytket-qujax
==================================

``pytket-qujax`` is an extension to ``pytket`` that allows ``pytket`` circuits to
be handed over to ``qujax``.

``pytket-qujax`` is available for Python 3.9, 3.10 and 3.11, on Linux and MacOS.
To install, run:

::

    pip install pytket-qujax
